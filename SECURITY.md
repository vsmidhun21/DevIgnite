## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

❌ Do NOT open a public issue.

Instead, contact:

* Email: [midhun21@zohomail.in](mailto:midhun21@zohomail.in) or [midhun890390@gmail.com](mailto:midhun890390@gmail.com)

Include:

* Description of the issue
* Steps to reproduce
* Possible impact
